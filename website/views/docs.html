About

Orchidei is an open-source file manager created by me - Atzuki , as a personal project. It is designedto allow users to share files across multiple devices quickly, easily and, mainly, for free.

Orchidei is a powerful and intuitive file manager which makes it easy to find, share and organize your files. It has many features designed to make life easier for users, including:

  • Easy navigation through folders and files
  • Ability to quickly transfer files between devices
  • Hash-protected files for security
  • Customizable settings for individual users

Orchidei is designed to be the file manager for users who need a modern and efficient way to manage their files. Give it a try today and see how Orchidei can make your file management tasks easier!

Installation

Orchidei is a file manager that can be used to manage your files and folders. Installing Orchidei is easy and can be done in just a few steps.

  1. Download the Orchidei installer. The installer can be found on the official Orchidei website.
  2. Once you have the installer, double click it to begin the installation process.
  3. You will be taken through an installation wizard. Follow the instructions on the screen. There shouldn't be any claims to accept, only the location of installation.
  4. Once you have completed the installation, Orchidei will be ready to use.

Congratulations, you have now installed Orchidei!

Settings

Welcome to Orchidei's Settings section! Here you can customize your experience with the Orchidei file manager.

To open the settings, type "orchsettings" into the search bar.

In the Orchidei Settings, you can modify the following parameters:

  • Language: You can select your preferred language from a range of options
  • Default Path: You can select the default folder path for your files
  • Safe Mode: You can activate or deactivate the Safe Mode feature for extra security used in deleting and copying files

Share files

Orchidei's file manager allows you to easily share your files across multiple devices. Here's how:

  • Right-click on the file you want to share and press the "Share" button.
  • A dialog box will open with additional settings, where you can set a limit on the number of downloads for the file before it is deleted.
  • Press "Share" and you will receive a secret hash. DO NOT SHARE THIS HASH as it will allow access to the shared file.

Share files

To receive a shared file, follow these steps:

  • Click on the folder where you want to save the file and press the "Receive" button.
  • A dialog box will open where you can paste the secret hash you received.
  • Press "Download" and the shared file will be saved to the selected folder.

In summary, Orchidei's "Share Files" feature allows you to easily share files with others and limit the number of downloads before it is deleted. And, to receive the shared file, paste the secret hash in the "Receive" dialog box and press "Download."
